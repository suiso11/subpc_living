# subpc_living

## 概要

サブPC上で動作するローカルLLM（Large Language Model）環境を構築するプロジェクトです。

## 目的

- クラウドサービスに依存しない、完全ローカルで動作するLLM環境を構築する
- サブPCのリソースを活用し、プライバシーを保ちながらAIを利用する
- 外部APIへの通信が不要なため、オフライン環境でも利用可能

## 特徴

- **ローカル実行**: すべての処理がサブPC上で完結
- **プライバシー重視**: データが外部に送信されない
- **カスタマイズ可能**: モデルやパラメータを自由に調整可能
- **低コスト運用**: クラウドAPI利用料が不要

## 環境

- サブPC上で動作
- ローカルLLMを使用
